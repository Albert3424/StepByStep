# Step By Step

Step by Step — это 2D игра-головоломка в жанре «сокобан», разработанная на Unity в рамках дипломного проекта.


## 🎮 Геймплей

Цель каждого уровня — поставить все кубики (ящики) на светящиеся целевые плиты. Игрок управляет персонажем, который может перемещаться по клеткам и толкать кубики.

Важные моменты:
- Кубик толкается только вперёд — тянуть его нельзя
- Нельзя толкнуть два кубика подряд — нужно планировать маршрут
- На 5 уровне после установки всех кубиков открывается дверь-выход — это завершает игру


## 🎯 Особенности

### Уровни
- 5 уровней с возрастающей сложностью
- Разные формы полей (прямоугольник, лабиринт, узкие коридоры)

### Ловушки
- Обычные шипы — мгновенная смерть
- Шипы с задержкой — активируются через 1 секунду после наступления
- Пустышки — ловушки-обманки, которые выглядят как шипы, но безопасны

### Прогресс и рекорды
- Игра сохраняет открытые уровни (нельзя перепрыгнуть)
- Сохраняются лучшие ходы и лучшее время для каждого уровня
- Рекорды отображаются в отдельной панели в главном меню

### Настройки
- Громкость (слайдер)
- Управление (Стрелки / WASD)


## ⌨️ Управление

- Движение:  ↑ ↓ ← → или W A S D (выбирается в настройках).
- Перезапуск уровня:  R 
- Выход в главное меню: кнопка в левом верхнем углу.


## 🛠 Стек технологий

- Движок: Unity 2022.3 LTS
- Язык: C#
- Уровни: Tilemap
- Интерфейс: Unity UI (uGUI)
- Сохранения: PlayerPrefs (локально)
- Звук: Audio Mixer, AudioSource
- Визуал: Пиксельные спрайты, шрифт Press Start 2P


## 🧪 Тестирование

Игра протестирована на Windows 10/11, разрешения экрана от 1280x720 до 2560x1440. Интерфейс адаптируется под любое разрешение.
